# Getting Started

This page is a placeholder for your first note. Replace it with your own
content, and add new entries to `src/SUMMARY.md` to make them appear in the
sidebar.

## Adding a new note

1. Create a Markdown file under `src/`, e.g. `src/notes/my-topic.md`.
2. Add a line to `src/SUMMARY.md` linking to it:
   ```markdown
   - [My Topic](./notes/my-topic.md)
   ```
3. Run `mdbook serve --open` to preview locally.

## Math reminder

Use `$ ... $` for inline and `$$ ... $$` for display math, e.g.

$$
e^{i\pi} + 1 = 0
$$
